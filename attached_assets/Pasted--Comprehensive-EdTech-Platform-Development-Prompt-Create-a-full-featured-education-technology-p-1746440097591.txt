*Comprehensive EdTech Platform Development Prompt*

"Create a full-featured education technology platform for TechieKraft that serves all academic subjects (STEM, humanities, arts, and languages) with the following specifications:

**Core Features:**
1. **Interactive Learning System**
   - Subject-specific modules with multimedia content
   - Virtual labs for science subjects
   - Writing workshops for humanities
   - Language learning tools with speech recognition
   - Math problem solver with step-by-step explanations

2. **Parent-Teacher Collaboration Suite**
   - Real-time messaging with read receipts
   - Automated progress reports with data visualization
   - Meeting scheduling with calendar integration
   - Behavior tracking and intervention system

3. **Assessment Tools**
   - AI-powered quiz generator
   - Automated grading for written responses
   - Plagiarism detection integration
   - Peer review functionality

**Technical Implementation:**
- **Backend (Django):**
  - Django REST Framework for API endpoints
  - PostgreSQL with optimized query performance
  - JWT authentication with role-based permissions
  - Celery for asynchronous tasks (email, report generation)
  - Redis for caching and real-time features

- **Frontend (Tailwind CSS):**
  - Responsive design with mobile-first approach
  - Accessible UI following WCAG 2.1 guidelines
  - Dark/light mode toggle
  - Interactive dashboards with Chart.js
  - Dynamic forms with HTMX for seamless interactions

**Integrations:**
- Google Classroom API for content synchronization
- Zoom/Google Meet API for virtual meetings
- Turnitin API for plagiarism checking
- Payment gateway for premium features
- SMS gateway for notifications

**Design Requirements:**
- **Color Scheme:**
  - Primary: TechieKraft Blue (#2563EB)
  - Secondary: Black (#111827) and White (#FFFFFF)
  - Accents: Blue variants (100-900 scale)

- **Typography:**
  - Headings: Montserrat (Bold/Semi-bold)
  - Body: Open Sans (Regular/Medium)
  - Code blocks: Fira Code
  - All fonts loaded via Google Fonts API

**Deployment & DevOps:**
- Docker containerization
- CI/CD pipeline with GitHub Actions
- AWS deployment (EC2, RDS, S3)
- Monitoring with Sentry and New Relic

**Quality Assurance:**
- Unit tests with pytest
- Integration tests with Selenium
- Load testing with Locust
- Security scanning with OWASP ZAP

**Deliverables:**
1. Complete source code with documentation
2. API documentation (Swagger/OpenAPI)
3. Deployment guide
4. User manuals for all roles
5. Test cases and reports

The platform should prioritize:
- Data privacy (GDPR/FERPA compliance)
- Scalability for institutional adoption
- Intuitive user experience across all devices
- Performance optimization for low-bandwidth scenarios

Include analytics capabilities for tracking engagement and learning outcomes, with customizable reports for administrators."
